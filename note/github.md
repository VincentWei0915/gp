- git remote add origin https://github.com/VincentWei0915/gp.git:連結雲端資料庫
- git branch -M main
    - branch：分支管理
    - -M：重新命名
    - main：主分支
- git push -u origin main：將檔案推到 main 分支
- git clone "URL"：將 github 整個資料夾載到本地(從0)
- git pull：將更改的檔案載到本地
- git checkout -b 新分支名稱
    - checkout：切換分支
    - -b：建立並切換至新分支\
